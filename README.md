# 🚀 Awesome RAG (Retrieval-Augmented Generation)

[![GitHub stars](https://img.shields.io/github/stars/RanaAhmar/awesome-rag)](https://github.com/RanaAhmar/awesome-rag/stargazers)

**Awesome RAG** is a curated list of the best resources, tools, research papers, and implementation patterns for Retrieval-Augmented Generation (RAG). 

RAG is the standard for building production-grade LLM applications that are grounded in your data. This repository helps you navigate the rapidly evolving landscape of vector databases, embedding models, and orchestration frameworks.
